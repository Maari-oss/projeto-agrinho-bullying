# Bullying nas escolas
Este projeto é uma página web informativa sobre os impactos do bullying nas escolas. Através de uma abordagem visual e clara, o conteúdo é dividido em seções que explicam os problemas causados pelo bullying nos adolescentes, suas consequências e as ações necessárias para mitigar seus efeitos.

### Estrutura do Projeto
Tecnologias Utilizadas:
HTML: Estrutura principal da página.
CSS: Estilização visual da página.
Fontes: Fontes personalizadas para melhorar a legibilidade e o design.
Arquivos:
index.html: Arquivo principal com a estrutura da página.
style.css: Arquivo de estilos para formatação da página.

### Descrição das Seções:
Cabeçalho (Header):
* Título da página com o nome "Bullying nas escolas".
* Parágrafo explicativo sobre como o bullying afeta os jovens.
* Navegação com links internos para as seções "Os Problemas", "Consequências" e "Conclusão".

#### Problemas:

Seção detalhando os principais problemas gerados pelo bullying:
O que o bullying causa nos adolescentes.
Ações que eles podem ter pela revolta de sofrer esse ataque.
Psicológico de quem provoca o bullying.

#### Consequências:

Lista das consequências diretas do desmatamento, como:
Baixa autoestima.A vítima pode passar a se sentir insegura e sem valor próprio.
Problemas de saúde mental.Pode desenvolver ansiedade, depressão e outros transtornos emocionais.
Dificuldades na escola.A concentração e o desempenho acadêmico podem piorar devido ao estresse.
Isolamento social.A vítima pode se afastar de amigos e atividades sociais para se proteger.
Risco de pensamentos suicidas.Em casos graves, o bullying pode levar a pensamentos de autoagressão.
Impacto a longo prazo.Os efeitos podem permanecer na vida adulta, afetando relacionamentos e bem-estar.

#### Conclusão:
Para resolver o problema do bullying, é essencial promover respeito, empatia e apoio, criando ambientes seguros e acolhedores para todos.
